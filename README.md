## Local Development

1. Clone this repository
2. Use a local server (Live Server extension in VS Code recommended)
3. Open `index.html`

## Usage

1. **Set Prices**: Configure material prices and exchange rates
2. **Manage Inventory**: Enter materials you already have
3. **Create Profiles**: Save different configurations for various scenarios
4. **Calculate Costs**: View real-time cost calculations with discounts

## Files Structure

- `index.html` - Main application
- `script.js` - Application logic
- `style.css` - Styling
- `meta_data.json` - Item definitions
- `profiles.json` - Profile storage (auto-generated)

## GitHub Pages Deployment

This app is designed to work with GitHub Pages. Simply push to your repository and enable GitHub Pages in settings.
